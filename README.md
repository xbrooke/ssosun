# 车载应用商店项目

## 本地开发

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 构建生产版本
```bash
npm run build
```

4. 预览生产版本
```bash
npm run preview
```

## 项目结构
- `src/components` - 可复用组件
- `src/pages` - 页面组件
- `src/data` - 静态数据
- `src/hooks` - 自定义hooks
- `src/lib` - 工具函数

## 技术栈
- React 18
- TypeScript
- Tailwind CSS
- Vite
